¿Qué es sitio web?

Un sitio web es un conjunto de páginas interconectadas que están disponibles en Internet y que sirven para compartir información, servicios, productos o contenido con usuarios de todo el mundo. Es la cara digital de una persona, empresa o proyecto, y puede adaptarse a diferentes propósitos, como informar, vender, educar o entretener.
